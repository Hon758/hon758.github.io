📌 Collepin

AI搭載型情報収集アプリ
気になる情報を、すぐにコレクション

アプリ概要

Collepinは、Webコンテンツを効率的に収集・整理・活用するためのAI搭載型情報管理アプリです。 従来のブックマークアプリを超えて、情報収集から活用までをシームレスに繋げる体験を提供します。

主要機能

🤖 AI自動処理

Gemini APIによる高精度なタイトル生成・要約・キーポイント抽出で、情報を自動的に整理します。

📌 簡単クリップ

Safari/Chromeからワンタップでクリップ。アプリ内でのURL入力による手動追加も可能です。

📁 柔軟な整理

フォルダ、カテゴリ、タグによる多軸分類。AI処理による自動カテゴリ・タグ提案機能付き。

🔍 高度な検索

全文検索とタグ検索。時系列・カテゴリ・タグ・完了状態による高度なフィルタリング機能。

👥 ソーシャル機能

フレンド機能とフォルダ共有による協力作業。リアルタイム同期で最新状態を共有。

🗺️ マップ表示

位置情報連携（Google Places API）による店名・住所・営業時間の自動抽出とマップ表示。

技術仕様

  • フロントエンド: Flutter (iOS/Android/Web/macOS対応)
  • バックエンド: Supabase (認証・データベース・ストレージ)
  • AI処理: Google Gemini 2.5 Flash-Lite API
  • 地図機能: Google Maps / Google Places API
  • 同期機能: リアルタイム同期とオフライン対応

プライバシーとセキュリティ

Collepinは、ユーザーのプライバシー保護を最優先に考えて設計されています。 収集した情報は適切に管理され、ユーザーの同意なく第三者に提供されることはありません。 詳細については、プライバシーポリシーをご確認ください。
